## Welcome to CirraUI

CirraUI is a modern, lightweight UI component library designed specifically for creators who want to build beautiful interfaces without the bloat.

### Why CirraUI?

- **Lightweight**: Minimal bundle size, maximum performance
- **Beautiful**: Carefully crafted components with attention to detail
- **Flexible**: Easy to customize and extend
- **Modern**: Built with the latest web technologies

### Getting Started

Browse through our component library to find the perfect components for your project. Each component is designed to be easy to use and fully customizable.

### How It Works

CirraUI is not a package you install. Simply browse the components, copy the code you need, and paste it into your project. Own your code, customize it however you want.

### Quick Example

Pick a component, copy the code, and you're ready to go:

```tsx
import { Button } from "@/components/ui/button";

export default function App() {
  return <Button variant="primary">Get Started</Button>;
}
```
